1 Cấu Trúc File Dự Án

Toàn bộ web nằm trong thư mục C:\web-truyen\. Các file quan trọng:

C:\web-truyen\
 ├── data\stories.json     ← ⭐ DỮ LIỆU TRUYỆN (thêm/sửa tại đây)
 ├── js\app.js           ← ⭐ Link affiliate + logic web
 ├── css\style.css        ← ⭐ Giao diện, màu sắc, font
 ├── index.html          ← Trang chủ
 ├── story.html          ← Trang chi tiết truyện
 ├── reader.html         ← Trang đọc truyện
 ├── guide.html          ← Trang hướng dẫn đăng truyện
 ├── images\covers\       ← Thư mục ảnh bìa (tự tạo)

Quy tắc: Mọi thay đổi nội dung → sửa data/stories.json. Thay đổi giao diện → sửa css/style.css. Sau khi sửa xong, chạy PUSH.bat để cập nhật web.

2 Đăng Truyện Mới

Mở file C:\web-truyen\data\stories.json bằng Notepad hoặc VS Code. Thêm một block truyện vào mảng JSON.

📋 Cấu trúc 1 truyện

// Mỗi truyện là 1 object trong mảng [...] của stories.json
{
  "id": "ten-truyen-khong-dau",     ← ID duy nhất, không dấu, dùng dấu gạch ngang
  "title": "Tên Truyện Đầy Đủ",
  "author": "Tên Tác Giả",
  "cover": "images/covers/ten-truyen.jpg",  ← đường dẫn ảnh bìa
  "genre": ["Tiên Hiệp", "Hành Động"],    ← mảng thể loại
  "status": "Đang ra",                    ← "Đang ra" hoặc "Hoàn thành"
  "rating": 4.8,
  "rateCount": 320,
  "views": 15000,
  "follows": 450,
  "source": "Tác giả tự viết",
  "description": "Mô tả ngắn về nội dung truyện, khoảng 1–3 câu.",
  "chapters": [
    {
      "id": 1,
      "title": "Chương 1: Khởi Đầu",
      "content": "Nội dung chương 1 ở đây...\n\nĐoạn 2..."
    },
    {
      "id": 2,
      "title": "Chương 2: Tiếp Theo",
      "content": "Nội dung chương 2..."
    }
  ]
}

📌 Giải thích từng trường

TrườngBắt buộcMô tả
idBẮT BUỘCMã định danh duy nhất. Dùng chữ thường, không dấu, thay space bằng -. VD: "kien-long-bat-bai"
titleBẮT BUỘCTên truyện hiển thị trên web
authorBẮT BUỘCTên tác giả
coverTùy chọnĐường dẫn ảnh bìa. Để trống hoặc bỏ qua nếu chưa có ảnh
genreBẮT BUỘCMảng thể loại. VD: ["Tiên Hiệp", "Hành Động"]
statusBẮT BUỘCChỉ nhận 2 giá trị: "Đang ra" hoặc "Hoàn thành"
ratingTùy chọnĐiểm đánh giá từ 1.0 đến 5.0
viewsTùy chọnSố lượt xem (hiển thị trên card)
descriptionBẮT BUỘCTóm tắt nội dung, hiển thị trên card và trang chi tiết
chaptersBẮT BUỘCMảng chương. Mỗi chương có: id (số), title (chuỗi), content (nội dung)

📝 Cách thêm truyện — từng bước

1
Mở file stories.json Dùng VS Code hoặc Notepad++. Đường dẫn: C:\web-truyen\data\stories.json
2
Tìm dấu ] cuối cùng của mảng File có dạng [{...}, {...}]. Thêm dấu phẩy sau truyện cuối cùng, rồi paste block truyện mới vào trước dấu ].
3
Kiểm tra JSON hợp lệ Dán nội dung vào jsonlint.com để kiểm tra lỗi cú pháp trước khi lưu.
4
Lưu và chạy PUSH.bat Lưu file → chạy PUSH.bat → web cập nhật trong 1–2 phút.
⚠️

Lỗi thường gặp: Thiếu dấu phẩy giữa 2 truyện, hoặc thừa dấu phẩy ở cuối mảng, sẽ khiến web không load được truyện nào. Luôn kiểm tra tại jsonlint.com trước khi push.

3 Sửa Ảnh Bìa Truyện

📁 Bước 1 — Chuẩn bị ảnh

  • Tạo thư mục: C:\web-truyen\images\covers\
  • Đặt file ảnh vào đó. VD: kiem-than-vo-song.jpg
  • Tỷ lệ lý tưởng: 2:3 (dọc). VD: 400×600px hoặc 300×450px
  • Định dạng: JPG hoặc WebP (nhỏ hơn, nhanh hơn)

✏️ Bước 2 — Gán vào truyện

Mở data/stories.json, tìm truyện cần sửa, đặt đường dẫn vào trường cover:

{
  "id": "kiem-than-vo-song",
  "title": "Kiếm Thần Vô Song",
  "cover": "images/covers/kiem-than-vo-song.jpg",
  ...
}
📌

Có thể dùng link ảnh ngoài thay vì file local. VD: "cover": "https://i.imgur.com/abc123.jpg" — tiện nếu ảnh đã có sẵn trên mạng.

🖼️ Ảnh bìa trên trang Hero Banner

Banner trang chủ tự động lấy ảnh từ truyện đầu tiên trong stories.json làm ảnh chính, và 2 truyện tiếp theo làm ảnh phụ bên phải. Muốn đổi ảnh banner → đổi thứ tự truyện (xem mục 5).

4 Sửa Ảnh Nền Hero Banner

Nền hero hiện dùng gradient màu xanh nhạt. Để thay bằng ảnh thật, mở css/style.css và tìm dòng .hero-banner:

🎨 Cách A — Đổi màu gradient

Tìm đoạn này trong css/style.css (khoảng dòng 80–90):

.hero-banner {
  background:linear-gradient(135deg,#bde0ef 0%,#d2eaf7 35%,#e8f5fc 65%,#f2f9fd 100%);
  min-height:320px; ...
}

Thay #bde0ef, #d2eaf7... bằng màu bạn muốn. Tham khảo màu tại coolors.co.

🖼️ Cách B — Dùng ảnh thật làm nền

1
Chuẩn bị ảnh nền Đặt ảnh vào C:\web-truyen\images\banner-bg.jpg. Kích thước khuyến nghị: 1920×600px.
2
Sửa CSS Thay toàn bộ dòng background:linear-gradient(...) trong .hero-banner bằng: 
.hero-banner {
  background: url('images/banner-bg.jpg') center/cover no-repeat;
  min-height:320px; position:relative; overflow:hidden;
  display:flex; align-items:center;
}
3
(Tùy chọn) Thêm overlay tối cho dễ đọc chữ Thêm vào sau class .hero-banner: 
.hero-banner::after {
  content: '';
  position: absolute; inset: 0;
  background: linear-gradient(to right, rgba(0,0,0,0.5) 0%, transparent 100%);
  pointer-events: none;
}
💡

Dùng ảnh có tone màu xanh-trắng, phong cảnh cổ trang để phù hợp với theme. Tìm ảnh miễn phí tại unsplash.com.

5 Sắp Xếp Thứ Tự Truyện

Truyện hiển thị trên trang chủ theo đúng thứ tự trong mảng stories.json. Truyện đứng đầu mảng = hiển thị đầu tiên, và cũng là truyện được đặt ở Hero Banner.

📋 Cách sắp xếp

Mở data/stories.json. File có dạng:

[
  { "id": "truyen-A", ... },   ← Hiển thị 1 - xuất hiện ở Hero Banner
  { "id": "truyen-B", ... },   ← Hiển thị 2
  { "id": "truyen-C", ... }    ← Hiển thị 3
]

Muốn đổi thứ tự → cắt (Ctrl+X) và dán (Ctrl+V) từng block truyện lên/xuống. Chú ý giữ đúng dấu phẩy.

⭐ Truyện nổi bật trong Sidebar

Sidebar "Truyện nổi bật" tự động sắp xếp theo số lượt xem (views) từ cao xuống thấp. Muốn một truyện lên top sidebar → tăng giá trị views của nó trong JSON.

📌

Muốn ghim một truyện cụ thể lên Hero Banner mà không phụ thuộc thứ tự → đặt truyện đó ở vị trí đầu tiên trong mảng JSON.

6 Gắn Link Affiliate

Khi người đọc nhấn "Chương tiếp", một popup xuất hiện hiện thị quảng cáo affiliate trong 5 giây. Cấu hình tại đầu file js/app.js.

📍 Vị trí trong code

Mở C:\web-truyen\js\app.js. Tìm ngay đầu file:

const AFFILIATE_ADS = [
  {
    title: "🛍️ Shopee Sale Hôm Nay",
    desc:  "Giảm đến 50% hàng ngàn sản phẩm – Click để xem ngay!",
    url:   "https://shope.ee/link-affiliate-cua-ban"       ← ĐỔI LINK NÀY
  },
  {
    title: "📚 Khóa Học Online Giảm 80%",
    desc:  "Học lập trình, thiết kế tại nhà. Ưu đãi có hạn!",
    url:   "https://link-affiliate-2.com"                   ← ĐỔI LINK NÀY
  },
  {
    title: "🎮 Nạp Game Giá Tốt Nhất",
    desc:  "Nạp thẻ game uy tín, giá rẻ, xử lý tức thì 24/7.",
    url:   "https://link-affiliate-3.com"                   ← ĐỔI LINK NÀY
  }
];

const INTERSTITIAL_SECONDS = 5;  ← đổi số giây đếm ngược (1–10)

✏️ Cách thêm/sửa quảng cáo

TrườngNội dung
titleBẮT BUỘCTiêu đề quảng cáo hiển thị to trong popup. Nên ngắn gọn, hấp dẫn, có emoji.
descBẮT BUỘCMô tả ngắn bên dưới tiêu đề. 1 câu kêu gọi hành động.
urlBẮT BUỘCLink affiliate đầy đủ. Khi người dùng click banner → mở link này trong tab mới.
1
Lấy link affiliate của bạn Đăng ký tại Shopee Affiliate, Lazada Affiliate, Accesstrade, hoặc bất kỳ mạng affiliate nào. Lấy đường link tracking cá nhân.
2
Mở js/app.js, thay URL Thay "https://shope.ee/your-affiliate-link-here" bằng link thật của bạn.
3
Thêm nhiều quảng cáo Thêm object mới vào mảng AFFILIATE_ADS. Hệ thống tự random chọn 1 ad mỗi lần chuyển chương.
💡

Tăng thu nhập: Đặt 5–10 quảng cáo khác nhau để không bị nhàm. Ưu tiên sản phẩm liên quan đến đối tượng đọc truyện: sách, game, app học tiếng Trung, khóa học viết lách...

7 Đẩy Cập Nhật Lên Web (Deploy)

Mỗi khi sửa bất kỳ file nào, cần đẩy code lên GitHub để web cập nhật.

🚀 Cách đơn giản nhất — dùng PUSH.bat

1
Lưu tất cả file đã sửa Ctrl+S trong VS Code hoặc Notepad.
2
Chạy PUSH.bat Double-click vào C:\web-truyen\PUSH.bat. Trình duyệt có thể mở yêu cầu đăng nhập GitHub — nhập tài khoản luulyhnc.
3
Chờ 1–2 phút GitHub Actions tự động build và deploy. Vào github.com/luulyhnc/luulyhiennhicac/actions để xem tiến trình.
4
Kiểm tra kết quả Mở https://luulyhnc.github.io/luulyhiennhicac/. Nếu vẫn thấy cũ → Ctrl+Shift+R để hard refresh.
📌

PUSH.bat tự động commit tất cả thay đổi. Không cần biết Git — chỉ cần chạy file bat là xong.

8 Chỉnh Sửa Giao Diện Khác

🎨 Đổi màu chủ đạo (teal → màu khác)

Mở css/style.css, tìm phần :root ở đầu file:

:root {
  --primary:   #3ab3ca;  ← màu chủ (nút, tab active, link...)
  --pri-d:     #2896ac;  ← màu tối hơn khi hover
  --pri-l:     #e0f3f8;  ← màu nền nhạt
  --bg:        #edf4fb;   ← màu nền trang
  --card:      #ffffff;   ← màu nền card/bảng
}

Chỉ cần đổi 3 dòng đầu (--primary, --pri-d, --pri-l) là toàn bộ web đổi màu.

🔤 Đổi font chữ

/* Dòng 1 trong style.css — đổi tên font trong URL */
@import url('https://fonts.googleapis.com/css2?
  family=Inter:wght@400;500;600;700&
  family=Merriweather:ital,wght@0,400;0,700;1,400;1,700&display=swap');

/* Body (giao diện) */
body { font-family: 'Inter', sans-serif; }

/* Nội dung đọc truyện */
.reader-body { font-family: 'Merriweather', Georgia, serif; }

📧 Đổi email nhận đơn đăng ký tác giả

Mở guide.html, tìm dòng:

const EMAIL = "admin@truyenhub.com"; ← đổi thành email của bạn

⏱️ Đổi thời gian đếm ngược affiliate

Mở js/app.js, tìm dòng:

const INTERSTITIAL_SECONDS = 5; ← đổi từ 1 đến 10
